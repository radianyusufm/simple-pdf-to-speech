Web ini menggunakan HTML, CSS, Bootstrap, JavaScript, dan AJAX pada bagian frontend. Pada

bagian backend, menggunakan Python Flask.

pertama, user mengunggah file PDF ke form. Kemudian, dari route "/index", Flask menerima request

POST dan menyimpan file PDF ke folder static,

kemudian file pdf di convert menjadi html menggunakan library PyMuPDF agar javascript bisa

mengakses isi teks pdf, dan agar tetap sederhana :) , lalu mengalihkan user ke route /upload.

Di frontend pada endpoint /upload, file PDF yang telah di convert menjadi HTML ditampilkan.

Ketika user memilih teks dan memilih bahasa, dan men-submit form, teks akan dikirim ke Flask

menggunakan AJAX.

Di Flask, pada route /upload ketika ada request POST, Flask akan menangkap teks tersebut, kemudian

menerjemahkannya sesuai dengan bahasa yang dipilih oleh user menggunakan library Googletrans.

Hasil terjemahan kemudian dikonversi menjadi audio dengan menggunakan library gTTS, dan

dikembalikan dalam format JSON.

Kemudian, dari AJAX, respons diterima dan ditampilkan menggunakan JavaScript, HTML. dan selesai.
